# Team AI Ethics Resource Hub

This is a static website for Team AI Session 3, focused on the ethics of AI production.
It adapts the Canvas lesson content into a shareable public resource for colleagues and students.

## Files

- `index.html`: the resource hub page
- `styles.css`: visual design and responsive layout
- `script.js`: menu, tabs, copy buttons, and quick usage reflection
- `assets/ai-ethics-production-hero.png`: generated hero image for the site

## How to preview

Open `index.html` in a browser.

## How to publish

This folder can be published as a static site through GitHub Pages, Netlify, or an institutional web server.

Before sharing broadly, consider:

- Replacing any private Canvas-only resources with public files.
- Checking whether embedded polls, videos, and H5P activities allow public access.
- Adding a reuse license, attribution language, and source notes.
- Adding additional AI literacy course materials as new pages or sections.
